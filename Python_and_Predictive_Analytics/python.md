**PYTHON & PREDICTIVE ANALYTICS**   

**Python Programming Course Description:**   
An introduction to the Python programming language. Work with python objects (including n-dimensional arrays), methods, basic visualizations, and user-defined functions. Connect to various data sources and create output files.  

**Predictive Analytics Course Description:**  
Utilize the Pandas library in Python to enable more advanced data manipulation and analysis. Implement machine learning algorithms to classify new data and predict unknown values. Perform basic time series analysis.
