**POWER BI & DATA VISUALIZATION**   

***PowerBI Course Description:***   
Utilize Power BI with dimensional modeling as a tool for business intelligence. Extract, transform, and load data from
multiple sources. Create calculated columns and measures for use in simple visuals (in matrices, tables, and
cards). Apply time intelligence functions and learn basic usage of common DAX commands.   

***Data Visualiztion Course Description:***   
Learn best practices for visualizing data depending on its type and the story being told. Create and share dashboards and
reports using Power BI and SQL Server Reporting Services. Build multi-page apps that feature interactivity and advanced
design elements..   

***Key Takeaways***   
Across these two courses, our primary focus was on developing proficiency in Power BI.  My initial impression of the tool was how impressive it is, particularly in how interactive and dynamic the dashboards can be.  Below you'll find our first assignment from the Data Visualization course, which particularly piqued my interest into the app.  In that class, we mainly focused on building reports and dashboards, working with filters and slicers, and incorporating features such as bookmarks, buttons, and drill-through functionality to enhance user experience.  We also spent the final week learning how to create paginated reports, which are designed to produce structured, print-ready outputs that present detailed data in a clear and organized format.

In the Power BI course, we focused on various data transformation techniques, including merging and appending datasets, pivoting and unpivoting data, and creating conditional columns.  We also explored data modeling concepts, such as understanding relationships between tables, creating hierarchies, and implementing row-level security.  Lastly, we worked extensively with calculated columns and DAX to build measures, perform advanced calculations, and generate meaningful insights from the data.  Provided below is our final project for the course, which required us to create a detailed report integrating the key concepts and techniques covered throughout the class.
